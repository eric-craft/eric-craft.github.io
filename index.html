Project: Machi Koro

For our senior capstone project at Wofford College, my group and I were tasked with creating a fully-functional console version of the popular tabletop game 'Machi Koro' using Java. We used a Scrum model, participating in daily scrums, two-week sprints, pair-programming, sprint reviews and sprint retrospectives. Our project consisted of hundreds of tests with high code coverage within a very extensive code base built from scratch.

The console playthrough of this Machi Koro game allowed for 'human vs. human' and 'human vs. AI' (strategic and non-strategic AI), and included a user guide and full API documentation.
